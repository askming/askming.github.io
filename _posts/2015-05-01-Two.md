Had a free afternoon today and decided to try creating a personal website using Github page and Jekyll. It's been a while since last time i tried and most are forgotten. This time, I decided to learn some basic knowledge about Jekyll so that I can customize my website.

**Some basics:**

- Each page can be written using markdown and saved as .md file, that is the content of the page
- The page formate is controlled by the layout variable specified in the heading for that markdown file
- The format files are stored in *_layouts* folder; each of the layout files uses the bootstrap files in *css,* *js* folders 
- After compiling, there will be a *_site* folder created for each web page (?)
