Concepts
___________

*fixed program computers*
*stored program computers*: treat data and instruction as the same thing

*interpreter*: legal set of instruction and consequently can be used to describe and accomplish anything you can with a computer.

a computer consists of: memory, control unit, arithmetic logic unit (accumulator)

six primitive instructions

*programming language*: a set of primitive instructions, a set of primitive (flow of) control structures.
 1.syntax: which sequences of characters and symbols constitute a well-formed string
 2.static semantics: which well-formed strings have a meaning
 3.semantics: what that meaning is
No ambiguity for each program (unique meaning).

Inappropriate program may leads to:
 1.crash
 2.never stop - infinite loop
 3.run to completion & produce the wrong answer 


### Compiled vs interpreted 
*interpreted*: source code - checker - interpreter - output (clear)
*compiled*: source code - checker/compiler - object code - interpreter - output (more efficient)
